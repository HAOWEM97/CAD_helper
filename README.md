# CAD 自动布线与通道配置 Web

基于 `React + TypeScript + Vite` 的 Local-First CAD 自动布线与通道配置工具。

当前版本已完成阶段 3：设备接线孔、常用库与路由代办，并已通过本地验证。

- 基础项目配置
- Redux 状态结构
- 非线性步骤导航
- 工作台四区布局
- PNG 底图导入与大图查看
- 左键标记参考点，右键或中键拖拽平移，滚轮缩放
- 两点参考点校准，参考点保存为底图像素坐标
- X/Y 独立像素到 CAD 转换比
- 状态栏 CAD 坐标显示
- 高倍率缩放黑屏修复，图纸放大后仍保持可见
- 本地草稿自动保存与页面刷新恢复
- 拓扑节点与通道连续绘制
- 正交锁定、节点吸附、轴线吸附
- 绘制 / 选择模式切换
- 节点拖动、节点与通道删除
- 通道类型编辑：线槽、排管
- 规格与敷设深度在阶段 4 前保持锁定
- 设备接线孔设置：设备实例、接线孔类型、安装高度、线缆组合
- 常用库：线缆规格、线缆组合、设备类型模板，工程内保存与浏览器全局复用
- 路由代办：有限线缆接线孔作为起点，不限接线孔作为承接终点
- 基于拓扑网络的二维最短路径路由和线缆兼容性校验
- 生成路由后高亮经过通道，并将线缆编号写回通道

下一阶段：阶段 4，规格推演与动态 BOM。
