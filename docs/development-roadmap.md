# Development Roadmap

开发严格按 `Plan.md` 的阶段推进。

## 当前阶段

阶段 3：设备接线孔、常用库与路由代办。

验收重点：

- 拓扑节点可以设置为设备接线孔，并保存设备实例、接线孔类型、安装高度和线缆组合。
- 自定义线缆规格、线缆组合、设备类型模板可保存到当前工程和浏览器全局库。
- 同类型设备默认使用“类型 + 最小空缺编号”命名，用户仍可自定义名称。
- 路由页按接线孔生成待办，不限数量接线孔只作为承接终点。
- 路由按 CAD 平面二维长度计算最短路径，并高亮经过通道。
- 网络断开时不生成错误路由。
