# CAD Script Spec

第一阶段 CAD 脚本输出目标为 `.bas` 文本文件，后续保留 `.scr` 输出结构。

## 输出原则

- 必须输出 CAD 坐标，不输出屏幕像素坐标。
- 线槽与排管应使用不同图层或线型表达。
- 排管需要包含规格文字标注。
- 未完成坐标校准或无有效拓扑时，不生成无意义脚本。

## 当前 `.bas` 实现

- 导出入口位于右侧“导出”页和顶部“导出 CAD 脚本”按钮。
- `.bas` 文件以 `DrawCadHelperChannels` 作为主过程。
- 脚本运行时创建或复用以下图层：
  - `CAD_HELPER_TRAY`：线槽。
  - `CAD_HELPER_DUCT`：排管。
  - `CAD_HELPER_ANNOTATION`：规格文字标注。
- 线槽使用 `AcadLWPolyline` 输出闭合空心边界轮廓，不使用 `ConstantWidth` 实体宽线。
- 线槽边界宽度取当前通道规格中的 `widthMm`；水平/垂直线槽拼接时会基于节点生成拼接块并输出并集外轮廓，避免转角、T 接或十字接处缺角。
- 排管使用 `AcadLine` 输出中心线，并在通道中点使用 `AcadText` 写入规格文字。
- 每条通道输出前写入注释，包含通道 id、类型和规格标签，便于和网页工程状态复查。
- `.scr` 当前作为生成器接口保留，可生成基础线槽、排管和文字命令，暂不作为主 UI 交付按钮。
