# 通道规格推荐规则

本文档记录当前阶段 4 已应用的线槽、排管规格推荐、用户确认、容积率校核和线缆分配规则。

## 规格确认

- 系统先根据有效路由中的线缆负载推荐通道规格。
- 推荐规格不是最终规格；用户点击“确认推荐”、选择常见规格或保存自定义规格后，才成为该通道的最终规格。
- 线缆负载变化后，系统保留用户已选最终规格，但标记为“需复核”，并重新计算容积率与警告。
- BOM 的通道规格统计优先使用最终规格；未确认时使用当前推荐规格。

## 线缆分类

- 接线孔明细中的用途包含“通信线”时，该线缆归为通信线。
- 其他用途都归为配电线，包括交流线、压缩机线、直流线、接地线、电源线等。
- 分类用于容积率校核和排管分配；具体用途仍会在明细中展示。

## 线槽规则

常见线槽规格：

- `200x150mm`
- `300x150mm`
- `500x150mm`

通信线和配电线同时存在时，推荐带分隔板线槽：

- `200x150mm（带分隔板150/50）`
- `300x150mm（带分隔板250/50）`
- `500x150mm（带分隔板450/50）`

容积率规则：

- 通信仓容积率低于 `50%`。
- 配电仓或普通线槽容积率低于 `40%`。
- 同时包含通信线和配电线但未使用分隔板时，显示警告。

## 排管规格

常见管型：

- `DN125 CPVC`：默认壁厚 `6mm`，内径 `113mm`。
- `DN100 CPVC`：默认壁厚 `5mm`，内径 `90mm`。
- `DN32 钢管`：内径 `37.4mm`。

常见组合：

- `16*DN125`
- `12*DN125`
- `8*DN125`
- `2*DN125+2*DN32`

## 排管分配

- 通信线必须走 `DN32`。
- 配电线可以走 `DN125`、`DN100`，也可以走空闲 `DN32`。
- `DN32` 被通信线占用后，不再分配给配电线。
- 相同线缆型号可以走同一根管道。
- 直流线和接地线可以走同一根管道。
- 交流线不能仅因为同属配电线就和直流线、接地线混管；除非同时满足“相同线缆型号”规则。
- 分配时优先使用大管，接近容积率上限时放入下一根兼容管。
- 缺少可独立敷设的管道时，显示明确警告。

容积率规则：

- 通信线在 `DN32` 中容积率低于 `50%`。
- 配电线在任意管道中容积率低于 `40%`。

## 展示规则

- 每根管道显示该管内线缆型号、用途、数量和该管容积率。
- 通道展开后的线缆明细按“型号 + 用途 + 外径 + 分类”合并统计显示。
- 总截面按当前通道内所有有效路由线缆汇总。
